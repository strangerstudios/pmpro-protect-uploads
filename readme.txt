Upload the `pmpro-protect-uploads` directory to the `/wp-content/plugins/` directory of your site.1. Activate the plugin through the 'Plugins' menu in WordPress.1. Files must be "attached" to WP posts to inherit the membership restrictions of the parent post.== Frequently Asked Questions === I found a bug in the plugin. =Please post it in the issues section of GitHub and we'll fix it as soon as we can.
